<h2><a href="https://www.geeksforgeeks.org/problems/smallest-factorial-number5929/1">Smallest number with at least n trailing zeroes in factorial</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given an integer <strong>n</strong>, find the <strong>smallest</strong> positive integer <strong>x</strong> such that the factorial of <strong>x</strong> (i.e., &nbsp;x! ) contains at least n <strong>trailing zeroes</strong>.</span></p>
<p><strong><span style="font-size: 14pt;">Examples:</span></strong></p>
<pre><span class="cf0" style="font-size: 18.6667px;"><strong>Input:</strong> n = 6<br></span><span class="cf0" style="font-size: 18.6667px;"><strong>Output:</strong> 25<br></span><span class="cf0" style="font-size: 18.6667px;"><strong>Explanation:</strong> 25! = 15511210043330985984000000, It ends with 6 trailing zeroes, which is the minimum required.</span></pre>
<pre><span class="cf0" style="font-size: 18.6667px;"><span class="cf0"><strong>Input:</strong> n = 1<br></span><span class="cf0"><strong>Output:</strong> 5<br></span><span class="cf0"><strong>Explanation:</strong> 5! = 120, It ends with 1 trailing zero, which meets the requirement.</span></span>
</pre>
